# E2E Testing

We use Maestro for E2E testing.

## Local setup

### 1. [Install maestro](https://docs.maestro.dev/getting-started/installing-maestro)

### 2. Run an E2E friendly version of the app

```
    # Run from the example folder
    npm run android:e2e
    # Or
    npm run ios:e2e
```

### 3. Run the E2E tests

```
    # Run from the example folder
    npm run e2e
```
